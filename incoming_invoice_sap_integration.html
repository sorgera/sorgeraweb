Incoming e-Invoice SAP Integration Package

With this solution, you can account for e-invoices received independently from the platform within the SAP system.

With the product, it is possible to access invoice images over journal entries, to track e-invoices that are not recognized, to establish authorization and approval structure between departments, to track history, to check current / amount / date / double entry entries.
